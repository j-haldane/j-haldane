Thief FP Controller

A player controller asset which behaves much like the old Dark Engine.

Features:

  • Climb any arbitrary mesh collider
  • Swimming
  • Leaning (Left, Right, Forward)
  • Object Interaction
  • Realistic Weapon Controller
  • Hide in Shadows
  • 3D Sound Diffraction System
  • Inventory System
